"Helsinki Cycling data analysis during years 2014-2019" is a study project of the Data Analysis by a Python training program. The datasets are located at the web site "Helsinki Region Infoshare". There are four parts implemented by four persons as below:
  Part 1: Data fetching and Data Cleaning.
  Part 2: Data grouping and Visualizing.
  Part 3: Data analysis among Helsinki measurement stations and the weather impact.
  Part 4: Data analysis within main traffic routes on Map.
