This folder includes Original Datasets from Helsinki, Vantaa and Espoo cyclists, which are Helsingin_pyorailijamaarat.xlsx, Espoon_pyorailijamaarat.xlsx and Vantaan_pyorailijamaarat.xlsx + Vantaan_pyorailijamaarat_2017.xlsx.

These Jupyter Notebooks is for cleaning datasets mentioned above: EspooBikers_Katariina.ipynb, HelsinkiBikers_Katariina.ipynb and VantaaBikers_Katariina.ipynb.

After clean up new dataset are stored here: EspooBikers_cleanDataset.csv, HelsinkiBikers_cleanDataset.csv and VantaaBikers_cleanDataset.csv.

My project Jupyter Notebooks is stored here with name Final_Project_Cyclists_by_Katariina.ipynb.
